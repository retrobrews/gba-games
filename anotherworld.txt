Another World GBA
-official conversion-

Converted to GBA by Cyril "Foxy" Cogordan (email: Foxy@Foxysofts.com / web: http://www.foxysofts.com)
Additionnal GBA gfx & musics by Eric Chahi
Original game by Eric Chahi & Jean-Francois Freitas.

Description:
Legendary adventure/plateforms game.

Digit sounds using Apex Audio System library (http://www.apex-designs.net)

Full working on real hardware.

How to play:
Use DPad to move
Button A to run
Button B to interact
Select to enter a code
Start to pause/unpause the game
Start+L+R to reset game

During pause:
DPad left & right to adjust brightness & contrast
